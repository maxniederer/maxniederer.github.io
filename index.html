LanguageQuest (2022)


The Gwitch'in people are a native tribe based in Alaska and territories of Canada. Their language is endangered with only 300 current speakers in the world.

This game aims to preserve the Gwitch'in culture through the art of play. The player can walk around a stylized Gwitch'in camp, talk with Gwitch'in-speaking NPCs, and do everyday tribe tasks like fishing and hunting.

This project was developed in the Unity Engine with a team of three other developers for the University of Pittsburgh.

CGI Maze (2020)


This algorithm generates a random maze on startup in 3d space, solves the maze in first-person camera using brute force, then solves the maze in the opposite direction using an optimized maze-solving algorithm.

This project was developed in C using OpenGL. All mathematical calculations were done from the ground-up without any external libraries.
